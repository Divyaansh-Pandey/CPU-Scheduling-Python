CPU Scheduling Algorithms in Python
Project Description

This project demonstrates the implementation of several CPU scheduling algorithms commonly used in operating systems. The algorithms included are:

First Come First Serve (FCFS)
Shortest Job First (SJF) – Non-Preemptive and Preemptive
Priority Scheduling
Round Robin
How to Run
Make sure Python is installed on your system.
Open a terminal or command prompt.
Navigate to the project directory.

Run the following command:
python cpu_scheduling.py
